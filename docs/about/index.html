What is Autoresearch?

Autoresearch is an autonomous research system. You define a research goal, and the system generates research questions, searches the web, writes findings with inline citations, and verifies everything through a three-judge review panel.

How It Works

goal.md defines the subject
  ↓
explode-goal.sh generates research tasks
  ↓
research.sh picks tasks, runs Claude Code (web search + write)
  ↓
3 judges review: Evidence, Consistency, Completeness
  ↓
PR merged → findings published → new tasks proposed
  ↓
Score climbs toward 100%
